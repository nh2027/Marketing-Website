# HEALIX Marketing Website

This is a static marketing website for HEALIX.

## Quick local preview

Open `index.html` directly in a browser, or serve the folder with a simple local server.

## Free hosting options

### GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub, go to **Settings > Pages**.
4. Set the source to deploy from the main branch.
5. Your site will be published on `https://yourusername.github.io/repository-name/`.

### Vercel / Netlify / Render
- Import the repository
- Build command: none required
- Publish directory: root folder

## Notes
- This site is static and does not require a backend.
- The demo email button in the CTA is placeholder text and can be changed.
